# Exercise Library

Explore and utilize ASAP's comprehensive exercise database.

## Overview

ASAP includes 1000+ pre-loaded exercises covering all major muscle groups and movement patterns. The exercise library helps you find the right exercises for your training goals.

## Browsing Exercises

### Exercise Tab

Access the library:

1. Go to **Exercises** tab in bottom navigation
2. Browse all available exercises
3. Use search and filters to find specific exercises

### Categories

Exercises are organized by muscle group:

- **Chest** - Bench press variations, flyes, dips
- **Back** - Rows, pulldowns, deadlifts
- **Shoulders** - Press variations, raises, upright rows
- **Arms** - Biceps curls, triceps extensions
- **Legs** - Squats, lunges, leg press, hamstring curls
- **Core** - Planks, crunches, ab wheel
- **Full Body** - Cleans, snatches, complexes
- **Cardio** - Running, cycling, rowing

### Search Function

Find exercises quickly:

1. Type in the search bar
2. Results filter in real-time
3. Search by exercise name or muscle group

## Exercise Details

When viewing an exercise, you'll see:

### Basic Information

- **Name** - Official exercise name
- **Category** - Primary muscle group
- **Description** - How to perform the exercise
- **Muscle Groups** - All muscles involved

### Your Stats

- **Personal Best** - Heaviest weight lifted
- **Total Volume** - Lifetime volume on this exercise
- **Last Performed** - Most recent session with this exercise
- **Frequency** - How often you perform it

### Exercise History

- List of all sessions including this exercise
- Sets, reps, and weights used each time
- Progress graph over time

## Using Exercises in Workouts

### Adding to Session

During a workout:

1. Click **"Add Exercise"**
2. Search or browse the library
3. Click the exercise to add it
4. Start logging sets

### Viewing History

Before a workout:

1. Find the exercise in the library
2. View your last performance
3. Plan to beat those numbers

## Filters and Sorting

### Filter by Category

1. Click category filter button
2. Select muscle group
3. View only exercises in that category

### Sort Options

Sort exercises by:

- **Alphabetical** - A to Z
- **Recently Used** - Exercises you've done recently
- **Most Frequent** - Exercises you do most often
- **Personal Best** - Highest weight achieved

## Exercise Variations

Many exercises have multiple variations:

**Bench Press:**

- Barbell Bench Press
- Dumbbell Bench Press
- Incline Bench Press
- Close-Grip Bench Press

**Squat:**

- Barbell Back Squat
- Front Squat
- Goblet Squat
- Bulgarian Split Squat

Choose the variation that matches your equipment and goals.

## Personal Records

Track your personal bests:

### Viewing PRs

1. Go to exercise details
2. See your all-time PR
3. View PR history over time

### Setting New PRs

PRs are automatically tracked when:

- You lift a heavier weight than before
- You hit more reps at a given weight
- You achieve higher total volume

## Exercise Notes

Some exercises include:

- **Form cues** - Key points for proper execution
- **Common mistakes** - What to avoid
- **Equipment variations** - Barbell, dumbbell, machine options
- **Progression tips** - How to advance the exercise

## Best Practices

### Choose Compound Movements

Prioritize these exercises:

- Squats, deadlifts, bench press, rows
- Train multiple muscle groups
- Build overall strength

### Track Consistently

Use the same exercise variations:

- Makes progress tracking easier
- Better data for analysis
- Clearer progression path

### Learn Proper Form

- Watch tutorial videos
- Start with lighter weights
- Add notes about form cues
- Focus on technique over weight

## Integration with Sessions

Exercises link to your training:

- See which sessions included the exercise
- Compare performance across sessions
- Track volume and intensity trends
