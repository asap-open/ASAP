# Profile & Stats

Manage your account settings, physical attributes, and view personal statistics.

## Overview

Your profile is the central hub for account management and personal data. It displays your physical attributes, training statistics, and provides access to account settings.

## Profile Information

### User Details

Your profile displays:

- **Full Name** - Display name
- **Username** - Login identifier
- **Email** - Account email
- **Member Since** - Account creation date

### Physical Attributes

Track important body metrics:

- **Height** - In centimeters
- **Current Weight** - Most recently logged weight
- **Target Weight** - Your goal weight
- **Gender** - Male, Female, or Other
- **Date of Birth** - For age-based calculations

### Body Metrics

Automatically calculated:

- **BMI** - Body Mass Index
- **BMI Category** - Underweight, Normal, Overweight, Obese
- **Weight Change** - Comparison to previous measurement

## Editing Your Profile

### Update Physical Attributes

1. Go to **Profile** tab
2. Click **"Edit Profile"**
3. Update any fields:
   - Full name
   - Height (cm)
   - Target weight (kg)
   - Date of birth
   - Gender
4. Click **"Save Changes"**

All fields are optional - update only what you need.

### Change Username

To change your username:

1. Go to **Profile** tab
2. Navigate to **User Management**
3. Select **"Account Settings"**
4. Enter new username
5. Confirm change

**Note:** Username must be unique across all users.

## Weight Management

### Logging Weight

Quick weight logging:

1. Click **"Log Weight"** button
2. Enter current weight in kg
3. Add optional note
4. Submit

This updates:

- Your profile's current weight
- Weight history
- BMI calculation
- Progress charts

### Weight History

View all weight logs:

- Chronological list

## Tips

### Keep Profile Updated

Regularly update:

- Current weight (weekly recommended)
- Target weight as goals change
- Height if measured precisely

### Track Body Metrics

Use profile data alongside training:

- Weight changes indicate calorie balance
- BMI helps track overall progress
- Compare weight to strength gains

### Set Realistic Targets

For target weight:

- Consider current weight and goals
- Allow for muscle gain during strength training
- Adjust target as you progress

## Integration with App

Your profile data powers:

- **Progress charts** - Body weight trends
- **BMI calculation** - Height + current weight
- **Analytics** - Personal statistics
- **Goal tracking** - Target weight progress

## Best Practices

### Regular Weigh-Ins

Consistent timing:

- Same time of day (morning recommended)
- Same conditions (fasted, post-bathroom)
- Once per week minimum

### Update as Needed

Keep profile current:

- Log weight consistently
- Update targets when goals change
- Maintain accurate physical data

### Security

Protect your account:

- Use strong passwords
- Don't share credentials
- Log out on shared devices
